# Retrieve an Invoice

Retrieves an Invoice object by ID.

## Endpoint

```
GET /v1/invoices/:id
```

## Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| id | string | yes | The ID of the Invoice |

## Returns

Returns an **Invoice** object including:
- `customer` — the **Customer** this Invoice belongs to
- `subscription` — the **Subscription** that generated this Invoice (if any)
- `charge` — the **Charge** created to pay this Invoice (once payment is attempted)
- `status` — `draft`, `open`, `paid`, `void`, `uncollectible`
- `amount_due` — total amount to be collected
- `lines` — line items (one per subscription item or one-time charge)

## Relationships

- An Invoice belongs_to a **Customer**
- An Invoice is generated by a **Subscription** billing cycle
- Payment triggers creation of a **Charge**
- A successful Charge sets Invoice `status` to `paid`, triggering `invoice.paid` **Webhook** event
- A failed Charge sets Invoice to `open`, triggering `invoice.payment_failed` **Webhook** event
- An Invoice can be manually voided (triggers `invoice.voided` event) or marked uncollectible
